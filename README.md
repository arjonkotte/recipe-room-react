# The Recipe Room

[See the app first!](https://recipe-room-react.herokuapp.com/)

# Overview

The recipe room is an app that is used to browse recipes and instrcutions added by various authors.

Technologies used (MERN stack):
* HTML
* Javascript
* CSS
* React
* Node.js
* Express
* MongoDB
* Mongoose
* Heroku
